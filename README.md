# 物理实验数据处理程序

## 项目依赖

> 使用pip下载

- toml
- numpy
- matplotlib
- sklearn



## 说明

使用演示：

```shell
$ python main.py 实验数据.toml
```

程序会输出计算结果文本。若实验需绘制曲线图，则图片输出到`pic`目录。

`template`目录里有全部实验的**数据模板**，且包含程序可接受的具体数据。
